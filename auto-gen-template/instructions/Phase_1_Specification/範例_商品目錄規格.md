# 商品目錄規格 範例

定義產品目錄的欄位、搜尋與排序需求，並提供 JSON Schema 範例。

## 欄位
- id: string
- name: string
- price: integer
- tags: string[]

```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "type": "object",
  "properties": {
    "id": {"type":"string"},
    "name": {"type":"string"},
    "price": {"type":"integer"}
  },
  "required": ["id","name","price"]
}
```
